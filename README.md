# Gabriela PM Portfolio

제품 전략과 실행을 함께 보여주는 개인 포트폴리오 웹사이트입니다.  
Vanilla HTML/CSS/JS로 제작한 정적 사이트이며, Product Manager로서의 프로젝트 경험, 문제 해결 방식, 그리고 결과를 간결하게 소개하는 것을 목표로 합니다.

## 프로젝트 소개

이 프로젝트는 PM 포트폴리오를 웹 형태로 정리한 사이트입니다.

- 핵심 프로젝트와 성과를 한눈에 보여주는 랜딩 구조
- Vanilla HTML/CSS/JS 기반의 가볍고 빠른 정적 페이지
- 추후 섹션 확장에 맞춰 재사용 가능한 디자인 토큰과 네비게이션 구조 적용

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript

## 로컬 실행 방법

별도의 패키지 설치나 빌드 과정 없이 바로 실행할 수 있습니다.

### 1. 저장소 클론

```bash
git clone <repository-url>
cd gabby-pm-portfolio
```

### 2. 프로젝트 실행

아래 방법 중 하나를 선택해 실행합니다.

#### 방법 A. 브라우저에서 `index.html` 직접 열기

```bash
index.html
```

#### 방법 B. 간단한 로컬 서버 실행

```bash
python -m http.server 5500
```

브라우저에서 `http://localhost:5500`으로 접속하면 됩니다.
